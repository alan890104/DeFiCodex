# DeFiCodex

## Setup

1. Download label dataset from [etherscan repo](https://github.com/brianleect/etherscan-labels)

2. create a virtual environment and install requirements

    ```bash
    python3 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    ```

3. Run the script

    ```bash
    python3 main.py
    ```
